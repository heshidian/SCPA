standard_flow_4 {SCPA}R Documentation

Standard flow

Description

Dimension reduction, clustering and clustering of single cell data were carried out.

Usage

standard_flow_4(
  sce,
  group = "orig.ident",
  dim_number = 20,
  dbs_rate = NULL,
  remove_doublets = TRUE,
  nmf_rank_number = 10,
  Scale_cellcycle = FALSE,
  resolution_low = 0.1,
  resolution_up = 1,
  resolution_by = 0.1,
  dir = "./3.scRNAflow"
)

Arguments

sce

Single cell Seurat data.

group

Grouping data of single cells.

dim_number

The number of dimension reduction.

dbs_rate

The expected doublet rate.

remove_doublets

Whether to remove doublet data.

nmf_rank_number

Choice of the factorization rank.

Scale_cellcycle

Remove cell cycle effects.

resolution_low

The minimum grouping parameter.

resolution_up

The maximum grouping parameter.

resolution_by

The interval of grouping parameters.

dir

Output data directory.

Value

a sce object
