quality_control_3 {SCPA}R Documentation

Data filtration

Description

Quality control and data screening for single cell data.

Usage

quality_control_3(
  sce,
  group = "orig.ident",
  nFeature_lower = 300,
  nFeature_upper = 8000,
  nCount_lower = 3,
  nCount_upper = 10000,
  percent_mito_lower = 0,
  percent_mito_upper = 25,
  percent_hb_upper = 10,
  percent_hb_lower = 0,
  percent_ribo_lower = 3,
  percent_ribo_upper = 1000,
  dir = "./2.Quality control"
)

Arguments

sce

Single cell Seurat data.

group

Grouping data of single cells.

nFeature_lower

The minimum number of genes detected per cell.

nFeature_upper

The maximum number of genes detected per cell.

nCount_lower

The minimum number of UMI per cell.

nCount_upper

The maximum number of UMI per cell.

percent_mito_lower

The minimum proportion of mitochondria.

percent_mito_upper

The maximum proportion of mitochondria.

percent_hb_upper

The maximum proportion of hemoglobin genes detected.

percent_hb_lower

The minimum proportion of hemoglobin genes detected.

percent_ribo_lower

The minimum gene expression ratio of ribosomal protein.

percent_ribo_upper

The maximum gene expression ratio of ribosomal protein.

dir

Output data directory.

Value

a sce object
