integrate_objects_2 {SCPA}R Documentation

Data integration

Description

Multiple single cell data were combined and integrated.

Usage

integrate_objects_2(
  sce,
  group = "orig.ident",
  input.type = c("list", "merged"),
  variablefeature.method = "vst",
  normalization.method.selection = c("LogNormalize", "SCT"),
  reduction.selection = c("cca", "rpca", "rlsi")
)

Arguments

sce

Single cell Seurat data.

group

Grouping data of single cells.

input.type

Single cell input type data

variablefeature.method

Variablefeature filtering method

normalization.method.selection

Standardized methods for single cell data

reduction.selection

Single cell data integration method

Value

a sce object
