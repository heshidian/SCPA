find_markers_5 {SCPA}R Documentation

Find markers

Description

Personalized gene visualization and differential gene selection.

Usage

find_markers_5(
  sce,
  group = "seurat_clusters",
  genes_list = c("IL7R", "CD3E", "CD8A", "CD3D", "CD2"),
  dir = "./4.Find markers"
)

Arguments

sce

Single cell Seurat data.

group

Grouping data of single cells.

genes_list

The expression value of selected genes was visualized.

dir

Output data directory.

Value

a sce object
